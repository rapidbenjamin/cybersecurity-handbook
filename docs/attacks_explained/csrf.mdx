## The solution

This attack can be mitigated by using a different and personalized token in each request. In nodejs, the [csurf](https://npmjs.org/package/csurf) library is highly recommended.

Implementation for:

- [Server render templates](https://www.npmjs.com/package/csurf#simple-express-example)
- [Ajax](https://www.npmjs.com/package/csurf#using-ajax)
- [Single Page Application (SPA)](https://www.npmjs.com/package/csurf#single-page-application-spa)
