## Basic Usage

### Download manually

```sh
git clone https://github.com/guidesmiths/cybersecurity-handbook
cd cybersecurity-handbook
```

### Setup

```sh
npm i
```

### Run

```sh
npm run dev
```

### Build

```sh
npm run build
```

### Serve built app

```sh
npm run serve
```
